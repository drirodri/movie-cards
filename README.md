### Rodando localmente

Clone o projeto

```bash
  git clone git@github.com:drirodri/movie-cards.git
```

Entre no diretório do projeto

```bash
  cd movie-cards
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run dev
```
